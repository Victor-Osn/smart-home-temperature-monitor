# Smart Home Temperature Monitor - Project Plan

## Project Goal
Build an IoT-style web application that monitors temperature readings and displays them through a simple dashboard.

## Main Features
- Show the current temperature
- Show recent temperature readings
- Simulate sensor data
- Display temperature status such as Normal, Warm, or Hot
- Simple and clean dashboard interface

## Project Structure
- Frontend: React
- Backend: Node.js and Express
- Data source: Simulated temperature readings
- Storage: In-memory data for now

## Why This Project
This project demonstrates frontend development, backend API creation, and an IoT-style dashboard without needing physical hardware at the beginning.
